Les langages utilisés

CSS3

CSS ou en francais "feuilles de style en cascade" est un langage de programmation qui décrit la présentation graphique d'un code HTML et XML.
Il est principalement utilisé pour la conception d'une page web.

HTML5

HTML5 ou HyperText Markup Language est un langage de programmation à balises qui utilise diverses balises pour spécifier le code. Il a été créé en 1992. La dernière version, HTML5, est utilisé depuis 2007 et est principalement utilisé pour la conception de sites web.

Javascript

Le javaScript est un langage de programmation web créé en 1995. Il a la particularité de s'exécuter du côté client et non serveur, le serveur envoie un code au client et le client exécute ce code.
il est très utilisé pour ajouter un effet esthétique et améliorer l'ergonomie et l'interface de la page web.

PHP

Hypertext Preprocessor ou plus connu sous le termes PHP est un language de programmation web principalement utilisé pour produire des sites internet dynamiques.
Souvent allié à un langage de gestion au base de données. Ne requiet aucun plugin ni logiciels pour fonctionner mais l'hebergeur doit prendre en compte ce langage.
contraire de Javascript, l'exécution est du côté serveur.

Les langages utilisés

Nous codons tous les sites de la même manière.

Nous avons une première phase de collecte des demandes du client.
On essaye au maximum de suivre les volontés du client pour que le site de ses rêves soit créé.

Ensuite, nous avons une deuxième phase, celle des maquettes. Nous créons les premières maquettes du site via plusieurs logiciels.

La troisième partie est le choix de l'hébergeur selon les volontés du client. Si le site doit intégrer du PHP ou non, la taille de la base de données,
le trafic des utilisateurs et plein d'autres données sont prises en compte, une sécurité anti-ddos est appliquée sur tous nos site.

Viens ensuite pour les offres supérieures la création de la base de données via un schéma, pour l'intégrer au mieux à la demande du client et éviter tout futur changement inutile.

Viens ensuite la partie code.
Le client est régulièrement consulté pour avoir son avis sur les avancements.

Pour finir vient la phase de mise en place et de test qui conclut la livraison, chacun de nos produits sont livrés avec un suivi de 6 mois pour toutes potentielles mises à jour.
